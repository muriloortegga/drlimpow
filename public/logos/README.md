# Logos Oficiais da Marca - Dr. Limpow

Insira nesta pasta os arquivos de logo oficiais da marca. O sistema está configurado para ler preferencialmente os arquivos SVG desta pasta se estiverem disponíveis.

## Arquivos Recomendados:

- **`logo-horizontal.svg`**: Logo em formato horizontal (ícone + texto "Dr. Limpow" ao lado), usado principalmente no Header.
- **`logo-vertical.svg`**: Logo em formato vertical (ícone acima, texto "Dr. Limpow" abaixo), usado em destaques ou telas dedicadas.
- **`logo-icon.svg`**: Apenas o ícone "rosto na gota", usado em locais pequenos ou como avatar/ícone de destaque.
- **`logo-tip.svg`**: Logo com a assinatura de serviços ("Higienização de Estofados e Automotiva"), usado principalmente no Footer.

*Caso você prefira usar arquivos rasterizados (PNG/JPG), salve-os como `logo-horizontal.png`, `logo-vertical.png`, `logo-icon.png` e altere a referência correspondente no componente `Logo.tsx`.*
